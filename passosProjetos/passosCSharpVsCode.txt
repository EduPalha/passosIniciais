Passos C# VsCode:

-Instalar asp.net 
-Digitar: dotnet new console --framework net7.0
-Substituir texto:

namespace ImersaoAVMB
{    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Hello World!");
        }
    }
}


-Com o script pronto, para executar o código basta digitar: dotnet run
-Adicionar pacotes/bibliotecas ao código: dotnet add package <PACKAGE_NAME> -v <VERSION>
-Listar pacotes/bibliotecas do código: dotnet list package
-Removendo pacotes/bibliotecas do código: dotnet remove package <PACKAGE_NAME>


Obs.: 
-> Dica: Baixar extensões no VSCode: C# (Microsoft), NuGet Package (aliasadidev);
